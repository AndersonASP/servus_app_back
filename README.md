# Servus Backend

Backend da aplicação Servus - Sistema de gestão de voluntariado.

## Tecnologias

- Node.js
- NestJS
- MongoDB
- TypeScript

## Instalação

```bash
npm install
```

## Execução

```bash
# Desenvolvimento
npm run start:dev

# Produção
npm run build
npm run start:prod
```

## Variáveis de Ambiente

Copie `env.example` para `.env` e configure as variáveis necessárias.
